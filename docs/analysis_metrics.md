# 分析指標の定義

YouTube収益化プロジェクトでは、練習生の強みやファンダムの構造を可視化するため、以下の独自の分析指標をGAS (`Code_export.gs`) で計算します。

## 1. コメント熱量スコア
コメントの質とエンゲージメントの深さを示す総合指標。以下の要素を組み合わせて算出します。

*   **計算式**: `コメント熱量スコア = (参加率 × 0.4) + (海外率 × 0.3) + (ポジティブ率 × 0.3)`
*   **構成要素**: 
    *   **参加率**: 動画視聴者数に対するコメント投稿者数の割合。コメントの広がりを示す。
    *   **海外率**: コメント投稿者のうち、海外からの投稿者の割合。グローバルなファンダムの広がりを示す。
    *   **ポジティブ率**: コメントの感情分析においてポジティブな感情を示すコメントの割合。ファンダムの熱量を示す。

## 2. その他の主要指標

| 指標名 | 定義 | 目的 |
| :--- | :--- | :--- |
| **コメントUU数** | ユニークユーザー数（Unique User）。コメントを投稿したユーザーの延べ人数。 | コメント参加者の規模を把握。 |
| **コメント分散率** | コメントが特定のユーザーに偏らず、広く分散しているかを示す指標。 | ファンダムの健全性、多様性を評価。 |
| **コミットメント率** | 動画視聴者数に対する、繰り返しコメントを投稿するユーザーの割合。 | 熱心なファンの割合を把握。 |
| **平均コメント数** | 1動画あたりのコメント総数。 | 動画コンテンツへの一般的な反応の多寡。 |
| **ジニ係数** | コメント投稿者の集中度合いを示す指標。0に近いほど均等、1に近いほど一部に集中。 | コメントの偏りを定量的に評価。 |
| **言語分布** | コメントに使用されている言語の割合。 | グローバルなファンダムの地域構成を把握。 |
| **感情分布** | コメントの感情（ポジティブ、ネガティブ、ニュートラル）の割合。 | ファンダムの感情傾向を把握。 |
| **エンゲージメント率** | 視聴回数に対するコメント数や高評価数の割合。 | 動画コンテンツへの反応の強さ。 |

## 3. 練習生タイプ別診断への適用
これらの指標は、Notionで定義された「練習生タイプ別診断（3軸8タイプ分類）」のロジックに組み込まれ、各練習生の特性を定量的に評価するために使用されます。特に「コメント熱量スコア」は、ファンダムの活性度を測る重要な指標となります。
