# 分析指標設計

## 1. 概要

YouTube Data API v3 から取得できる公開データを基に、アイドルの強みとファンダムの構造を可視化するための指標を定義します。本プロジェクトでは、PRODUCE 101 JAPAN 新世界 (SHINSEKAI) のチッケム動画（推しカメラ）を主な分析対象とします。

## 2. 基本指標

### 動画パフォーマンス指標

| 指標名 | 計算式 | 説明 |
|---|---|---|
| 再生回数 (viewCount) | API取得値 | 動画の総再生回数 |
| いいね数 (likeCount) | API取得値 | 動画の総いいね数 |
| コメント数 (commentCount) | API取得値 | 動画の総コメント数 |
| エンゲージメント率 | `(likeCount + commentCount) / viewCount × 100` | 視聴者の能動的関与度（%） |
| いいね率 | `likeCount / viewCount × 100` | 視聴者のポジティブ反応率（%） |
| コメント率 | `commentCount / viewCount × 100` | 視聴者の発言参加率（%） |

### コメント分析指標

| 指標名 | 説明 |
|---|---|
| 総コメント数 | 動画に投稿されたコメントの総数 |
| ユニークコメンター数 | 重複を除いたコメント投稿者数 |
| リピートコメンター率 | `ユニークコメンター数 / 総コメント数` の逆数（ファンダム密度の指標） |
| コメントいいね合計 | コメント自体のいいね数の合計（共感度の指標） |

## 3. プレイリスト別分析

`playlist_id` 列を活用して、以下のプレイリスト別分析が可能です。

| プレイリスト | 分析目的 |
|---|---|
| Theme Song THE FINAL CLOSE-UP 💙 | 最新動画の推移追跡・ファン反応分析 |
| 推しカメラ｜新世界 (SHINSEKAI) | チッケム動画のパフォーマンス比較・練習生別強み分析 |
| SHINSEKAI SELFIE CHALLENGE 📷 | ファンエンゲージメント・自撮り動画の反応分析 |
| SHINSEKAI 1MIN PR 🌏 | 自己PR動画の訴求力分析 |

## 4. ファンダム分析指標（STAT PICK独自指標）

### ファンタイプ分類

コメント行動パターンに基づき、ファンを以下のタイプに分類します。

| タイプコード | タイプ名 | 定義 |
|---|---|---|
| `ACTIVE` | アクティブファン | 複数動画に複数コメントを投稿するコアファン |
| `DEEP` | ディープファン | 少数動画に深くコメントするニッチ支持者 |
| `WATCHER` | ウォッチャー | いいねのみでコメントしない観察型ファン |
| `CASUAL` | カジュアルファン | 1〜2回のみコメントする一般視聴者 |

### ファンダム構造指標

| 指標名 | 計算式 | 説明 |
|---|---|---|
| コアファン比率 | `ACTIVEファン数 / 全ユニークコメンター数` | ファンダムの結束度 |
| ファン多様性スコア | シャノンエントロピーで計算 | ファンタイプの分散度 |
| クロスプレイリスト率 | 複数プレイリストにコメントしたユーザー比率 | ファンの幅広い関与度 |

## 5. 時系列分析

`daily_log` シートのデータを活用した時系列分析。

| 指標名 | 説明 |
|---|---|
| 日次再生増加数 | 前日比の再生回数増加量 |
| 日次コメント増加数 | 前日比のコメント数増加量 |
| バイラル係数 | 短期間での急激な再生増加率（話題性の指標） |
| 減衰率 | 公開後の再生数の減少速度（コンテンツ持続力の指標） |

## 6. 注意事項・制約

- YouTube Data API v3 の公開データのみを使用するため、非公開情報（視聴者の属性、正確なインプレッション数など）は取得不可。
- コメント数は YouTube 側の仕様により、実際の投稿数と若干の誤差が生じる場合がある。
- コメントが無効化されている動画は、コメント分析の対象外となる。
- APIクォータ制限（1日10,000ユニット）に注意。大量の動画を処理する場合は分割実行を検討する。
- 同一動画が複数プレイリストに含まれる場合、`playlist_id` は優先度の高いプレイリストのIDが設定される。
